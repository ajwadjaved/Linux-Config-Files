Currently contains the wallpaper/lockscreen backgrounds I use (mostly 1920x1080), the aliases I have in .bashrc and a minimalistic python ide in the .vimrc
  
Vimium shortcuts:

```
map H scrollLeft
map L scrollRight
map h goBack
map l goForward

map x restoreTab
map X removeTab

map w removeTab
map W restoreTab

map n removeTab
map N restoreTab

map , visitPreviousTab
map . Marks.activateGotoMode

unmap /
```
